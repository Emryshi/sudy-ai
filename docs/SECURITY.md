# Güvenlik Politikası

## Desteklenen Sürümler

Şu anda yalnızca en son kararlı sürüm için güvenlik yamaları sağlanmaktadır.

| Sürüm | Destek Durumu |
| ----- | ------------- |
| 0.1.x | Destekleniyor  |
| < 0.1 | Desteklenmiyor|

## Güvenlik Açıklarının Bildirilmesi

Sudy projesinde bir güvenlik açığı tespit ederseniz, lütfen bunu genel bir GitHub Issue olarak açmak yerine, doğrudan proje yöneticilerinin e-posta adresleri üzerinden veya özel bildirim kanallarından gizli bir şekilde bize bildirin. Bu sayede açığı kapatmak için bir yama yayınlayana kadar kullanıcılarımızı korumuş oluruz.

Bildirilen açıklar 48 saat içinde değerlendirilecek ve en kısa sürede düzeltilerek yeni bir yama sürümü yayınlanacaktır.
